# Como aplicar a correção

## 1. Subir no GitHub

No repositório, substitua estes arquivos:

- index.html
- 404.html
- supabase-config.js

Faça commit.

## 2. Esperar o GitHub Pages atualizar

Aguarde alguns minutos e abra o site de novo.

## 3. Testar

1. Entre como treinador.
2. Selecione a aluna real do Supabase.
3. Abra o plano gerado pela IA.
4. Os exercícios devem aparecer automaticamente.
5. Se não aparecerem de primeira, clique no botão `Sincronizar Supabase`.

## Importante

Não precisa rodar o SQL de novo se ele já foi executado com sucesso.
Não precisa recriar a Edge Function se a geração da IA já está funcionando.
